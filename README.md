# Greenwich Commute

A clean, real-time commute dashboard for journeys from Greenwich, London.

## Live webpage

**[Open the Greenwich Commute dashboard](https://arashnass.github.io/greenwich-commute/)**

## Overview

Greenwich Commute brings useful travel information into one simple page, helping users quickly review conditions before leaving home or work.

The dashboard is designed around:

- Current time and local weather
- Greenwich transport information
- Clear service-status indicators
- Fast, mobile-friendly access
- A simple interface inspired by Transport for London and National Rail styling

## Technology

The project is built as a lightweight standalone webpage using:

- HTML
- CSS
- JavaScript

No installation or build process is required.

## Run locally

Download or clone the repository, then open `index.html` in a web browser.

```bash
git clone https://github.com/ArashNass/greenwich-commute.git
cd greenwich-commute
```

You can also serve it locally with a simple web server:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.
